# 映射
## 概念
1. 映射：map，通过键值对相连
2. 键值对：一对匹配的信息
3. 例：学号(键 key)-姓名(值 value)
## 声明 赋值
- key与value的类型可以是 bool string 数字 指针 channel
- key通常为int与string类型
- value通常为数字，string，map，结构体
```Golang
方式一
var m map[int]string//先声明
m = make(map[int]string,10)//空映射,创造一个空间//再make一个空间
m[202400204042] = "小白"
m[202400204043] = "小黑"
fmt.Println(m)
```

```Golang
方式二
b := make(map[int]string)
b[202400204042] = "小白"
b[202400204043] = "小黑"
fmt.Println(b)
方式三
c := map[int]string{
    202400204042 : "小白",
    202400204043 : "小黑",
    }
fmt.Println(c)
```
##  注意事项
- map集合使用前一定要make
- map的key-value对是无序的，不能通过下标访问，只能通过键访问
- value部分可以重复，key不能重复
- make函数的第二个参数size，表示映射的容量，即可以存储多少个键值对，默认值为0，表示无限容量,可以省略
## 其他操作
```Golang
m := map[int]string{
        1 : "张三",
        2 : "李四",
    }
    
//增加与更新
map[key] = value

//删除
delete(m, key)

//清空 == 遍历+逐个删除
for k := range m {
        delete(m, k)
    }
//或者make一个新map
m = make(map[int]string)
    m[2] = "奥创"
    fmt.Println(m)
    
//查找
value,exist := m[2]
fmt.Println(value,exist)

//for range遍历
for key,value := range m {
	fmt.Println(key,value)
}//只有for range遍历，不能for遍历
```
## 嵌套映射
```Golang
a := make(map[string]map[int]string)
//嵌套map
//value为map[int]string
a["法师"] = map[int]string{102:"甄姬",202:"高渐离"}
    for key,value := range a {
        fmt.Println(key)
        for key2,value2 := range value {
            fmt.Println(key2,value2)
        }
    }
```
