# 获取用户终端输入
>[!success]- Scanln: 读取用户输入，直到遇到空格或换行符
>```Golang
>var name string
  var age int
  var score float32  fmt.Scanln(&name,&age,&score)
fmt.Println("name:",name,"age:",age,"score:",score)

>[!success]- Scanf: 读取用户输入，需要指定格式
>```Golang
>fmt.Scanf("%s %d %f",&name,&age,&score)
fmt.Println("name:",name,"age:",age,"score:",score)
