# 代码注意事项
- 每条语句结尾不需要加分号（会自动补充）但是是在语句结尾加分号且只加一个
- 定义的变量与import的包没有使用到的话，代码不能编译通过
- 大括号成对出现
![[0.Hello World！]]
# 注释
- 行注释：ctrl + /
- 块注释：shift + Alt + a
# 代码风格
- 注意缩进：tab往后缩进，shift + tab向前缩进
- 一行语句不超过80个字符，若输出超过80个字符可以分段，将每一段用双引号标注，之间用逗号隔开并换行
