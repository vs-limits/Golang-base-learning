# 切片基础
- 切片是对数组一个连续片段的引用，可以方便的对数组进行操作，比如添加、删除元素、修改元素等。
## 声明
```Golang
s := make([]int, 5)

var s []int = make([]int, 5)
```
## 定义 赋值
```Golang
方式一
定义一个切片，让切片引用一个已经创建好的数组
var intarr [5]int = [5]int{1, 2, 3, 4, 5}//数组
    var intslice []int = intarr[1:3]//从1开始到4结束，不包括4
    
该方式容量与引用数组的容量相等

```

```Golang
方式二
通过make内置函数创建切片
var slice2 = make([]int,5,16)//5是长度，16是容量
make底层创建一个数组，不可以直接操作这个数组，要通过slice间接的访问各个元素
例：
slice2[0] = 26

```

```Golang
方式三
定义一个切片，直接就指定具体数组
slice3 := []int{1,4,7,8}
该方式的容量与长度相同
```
## 遍历
- for 循环遍历
```Golang
slice0 := []int{66,88,99}
for i := 0;i < len(slice0);i++ {
      fmt.Println(slice0[i])
      fmt.Printf("slice0[%v] = %v\n",i,slice0[i])
    }
```
- for range遍历
```Golang
for i,value := range slice0 {
        fmt.Printf("slice0[%v] = %v\n",i,value)
    }
```
- 注意事项
>[!success]- 
>1.切片定义后不能直接使用，需要让其引用一个数组，或者make一个空间供slice使用
>2.切片不能越界

>[!success]-
> 切片可以动态增长，但是不能缩短，只能在末尾添加元素，或者使用append函数添加元素
> ```Golang
> slice1 := append(slice0,55,22)
    slice0 = append(slice0,77)
    fmt.Println(slice1)
    fmt.Println(slice0)
    
1. 底层追加元素，是对数组进行扩容，老数组变新数组
2. append函数返回一个新的切片，原切片不变
3. 底层数组的指针指的是新数组

```Golang
可以通过append函数将切片追加给另一个切片
slice1 := append(slice0,55,22)
slice0 = append(slice0,77)
slice2 := append(slice1,slice0...)//在slice1的基础上追加slice0
    fmt.Println(slice2)
```
- 拷贝
```Golang
var a []int = []int{2,5,8,11,9}
    var b []int = make([]int,3)
    //拷贝
    copy(b,a)//不论n有多大，都只拷贝a的前n个元素到b
    fmt.Println(b)
```
