# 数组基础
## 定义 赋值
```Golang
//定义
var arr [5]int

//赋值
arr[0] = 89
```
## 数组遍历
- for循环
```Golang
//输入数据 填入数组
for i := 0;i < len(arr);i++ {
	   fmt.Scanln(&arr[i])
    }
//遍历数组
for i := 0;i < len(arr);i++ {
        fmt.Println("数组元素", i, "是：", arr[i])
    }
```
- for range循环
```Golang
//普通调用
for key , value := range arr {
        fmt.Println("数组元素", key, "是：", value)
    }
//省略索引
for _,value := range arr {
        fmt.Println("数组元素", value)
    }
```
## 初始化方式
- 方式一    完整方式
```Golang
    var arr1 [3]int = [3]int{1,2,3}
```
- 方式二    直接初始化
```Golang
var arr2 = [3]int{1,4,7}
```
- 方式三    省略长度
```Golang
var arr3 = [...]int{1,4,7,11}//输入任意个数
```
- 方式四    指定索引值
```Golang
var arr4 = [...]int{2:66,0:33,1:0}
```
- 注意事项
1. 数组的长度属于类型的一部分
2. 数组的长度必须是常量或者变量，不能是表达式
3. 数组默认下是值传递，值拷贝，所以修改数组元素的值不会影响到原数组
4. 要在其他函数中改变原来数组，可以使用引用传递
**例**：
```Golang
test2(&arr5)

func test2(arr *[3]int) {
    arr[0] = 7
}//这样才能改变原数组
```
## 多维数组
- 二维
```Golang
var arr [2][2]int
```
- 三维
```Golang
var arr [2][2][2]int
```
- 多维
```Golang
var arr [2][2][2][2]...int//几维就几个
```
- 数组的指针都是指向第一个位置的地址
**例**：
```Golang
&arr == &arr[0] == &arr[0][0]
&arr[1] == &arr[1][0]
```
- 多维数组的初始化
```Golang
var arr2 [2][3]int = [2][3]int{{1,3,5},{2,4,6}}
```
## 遍历二维数组
- for循环
```Golang
for i := 0; i < len(arr2); i++ {
        for j := 0;j < len(arr2[i]);j++ {
           fmt.Println(arr2[i][j])
        }
    }
```
- for range循环
```Golang
for index,arr01 := range arr2 {
        for _,value := range arr01 {
           fmt.Println(index,arr01,value)
        }
    }
```
